Allows you to create and organise lobbying and appeal campaigns.


== Description ==

**WARNING**: This project is still in development and is currently not ready for production use. Please install at your own risk.

Activists without Lobbies makes it possible to start and manage political lobbying campaigns (e.g. UKUncut, 38Degrees). By using libraries such as MySociety, AwL allows you to collect signatures, email MPs, inform supporters, etc all from within the simplicity of WordPress.

== Installation ==

1. Upload `activists-without-lobbies` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make any changes the to provided template files and store them in your theme directory
